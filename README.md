# Professional Task Optimizer
Ek simple lekin powerful Python-based task management system.

## Introduction
Yeh project users ko unke daily tasks manage karne aur unki productivity track karne mein madad karta hai.

## Features
* **Task Addition:** Naye kaam add karein.
* **Status Tracking:** Dekhein kaunsa kaam baaki hai aur kaunsa khatam.
* **Data Persistence:** Tasks ko local file mein save karna.
